\# Controlled Impedance \& Stackup Specification



\## Dielectric Material Reference

\* \*\*Core \& Prepreg Layer Material:\*\* FR-4 High-Tg (Tg >= 170°C)

\* \*\*Dielectric Constant (Er):\*\* 4.2 at 1 GHz target analysis



\## Trace Geometry Layout Rules

\* \*\*USB 2.0 Diff Pair (90 Ohm Target):\*\* Width 0.18 mm, Spacing 0.15 mm on Layer 1.

\* \*\*Single-Ended Digital (50 Ohm Target):\*\* Width 0.25 mm on Layer 1.
